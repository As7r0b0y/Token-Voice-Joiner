 # Como usar
 * Insira o ID do servidor que as Tokens estão dentro.
 * Após isso Insira o ID da chamada de voz que as Tokens iram entrar.
 * Lembre-se de ver as permissões das Tokens para conectarem na chamada de voz.   
   
 # Requerimentos
 * pygame
 * colorama
 * pystyle
 * websocket
 * time
 * json
 * concurrent.futures
 * os
